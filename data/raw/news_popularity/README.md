# Online News Popularity (Products / Articles)

**Source**: UCI ML Repository (id=332)
**Entity column**: `url (article identifier)`
**Use case for this pipeline**: Cluster articles by content style and channel; predict popularity tier

## Description
39,797 Mashable.com articles. 60 numeric features describing the article: word counts, links, images/videos, channel, day of week, keyword polarity scores, plus the share count (popularity target).

# Wine Quality red + white (Products / Sensory)

**Source**: UCI ML Repository (id=186)
**Entity column**: `row index (one row per wine sample)`
**Use case for this pipeline**: Cluster wines by chemistry profile; correlate with quality score

## Description
6,497 Portuguese 'Vinho Verde' wine samples. 11 physicochemical features (fixed acidity, volatile acidity, citric acid, residual sugar, chlorides, free SO2, total SO2, density, pH, sulphates, alcohol) + quality (0-10) + type (red/white).

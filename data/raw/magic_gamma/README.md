# MAGIC Gamma Telescope (Signals / Physics)

**Source**: UCI ML Repository (id=159)
**Entity column**: `row index (one row per Cherenkov burst measurement)`
**Use case for this pipeline**: Cluster gamma-ray detector signal windows; 2 ground-truth classes (gamma signal vs hadron background) hidden in 10 physical signal descriptors

## Description
19,020 high-energy gamma-ray detector measurements from a Cherenkov telescope. 10 numeric features: shower length, width, size, concentrations, asymmetries, plus class label. Classic signal-vs-noise benchmark on real physics data.

# Wholesale Customers (Product categories)

**Source**: UCI ML Repository (id=292)
**Entity column**: `row index (one row per wholesale customer)`
**Use case for this pipeline**: Cluster wholesale buyers by annual spend across 6 product categories

## Description
440 wholesale customers of a Portuguese distributor. Features: Fresh, Milk, Grocery, Frozen, Detergents_Paper, Delicassen annual spend (monetary units). Classic product-preference segmentation benchmark.

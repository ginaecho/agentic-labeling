# Bank Marketing (Customer + Campaign)

**Source**: UCI ML Repository (id=222)
**Entity column**: `client ID (row index)`
**Use case for this pipeline**: Cluster customers by demographics and campaign-response behaviour

## Description
41,188 records from a Portuguese bank's telemarketing campaigns. Features: age, job, marital status, education, balance, call duration, number of contacts, previous campaign outcome, subscription outcome (y).

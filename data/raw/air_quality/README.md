# Air Quality UCI (Sensor data)

**Source**: UCI ML Repository (id=360)
**Entity column**: `sensor location (implicit, single site)`
**Use case for this pipeline**: Cluster hourly sensor reading patterns across the day/week

## Description
9,358 hourly readings from 5 chemical sensors in an Italian city (2004-2005). Features: CO, NMHC, NOx, NO2, O3 sensor responses, temperature, relative humidity, absolute humidity.

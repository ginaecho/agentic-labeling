# Online Retail II (E-commerce transactions)

**Source**: UCI ML Repository (id=352)
**Entity column**: `CustomerID`
**Use case for this pipeline**: Aggregate per-customer RFM features (recency, frequency, monetary) to cluster buyer personas

## Description
1,067,371 transactions from a UK online retailer (2009-2011). Each row is one invoice line: CustomerID, StockCode, Description, Quantity, UnitPrice, Country.

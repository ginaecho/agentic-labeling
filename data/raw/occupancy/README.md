# Occupancy Detection (IoT / Sensor)

**Source**: UCI ML Repository (id=357)
**Entity column**: `Timestamp (time-windowed rows)`
**Use case for this pipeline**: Cluster building occupancy patterns by environmental sensor profile

## Description
20,560 readings from a room equipped with environmental sensors. Features: Temperature, Humidity, Light, CO2, HumidityRatio, Occupancy (ground truth label).

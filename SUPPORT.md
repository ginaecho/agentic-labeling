# Support

> **Note:** This is a personal research project, **not** an official Microsoft
> product. It is provided as-is, with no guaranteed support or service-level
> agreement.

## How to Get Help

- **Questions & ideas:** Open a [GitHub Discussion](https://github.com/tzuchunchen_microsoft/Agentic_Labelling/discussions) (if enabled) or a GitHub issue with the `question` label.
- **Bugs:** Open a [GitHub issue](https://github.com/tzuchunchen_microsoft/Agentic_Labelling/issues) with clear reproduction steps, your OS, and Python version.
- **Feature requests:** Open a GitHub issue describing the use case and motivation.
- **Security issues:** Do **not** open a public issue — see [SECURITY.md](SECURITY.md).

## Before Filing an Issue

1. Check the [README](README.md), especially the [Quick Start](README.md#-quick-start) and [Configuration](README.md#️-configuration-configyaml) sections.
2. Search [existing issues](https://github.com/tzuchunchen_microsoft/Agentic_Labelling/issues) to avoid duplicates.
3. Confirm the problem reproduces on a fresh clone with the pinned dependencies in `requirements.txt`.

## Response Expectations

Maintainers respond on a best-effort basis. Because this is a research project,
response times may vary.
